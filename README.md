# 📚 Smart Book Creator

Smart Book Creator is a full-stack AI-powered web application that allows users to generate structured books using AI. Users can create book outlines, generate chapters, edit content, and export books in multiple formats.

The system uses a React frontend, Node.js/Express backend, MongoDB database, and integrates with Google Gemini AI for content generation.

## ⚙️ Tech Stack

### Frontend
- React (Vite)
- React Router
- Axios
- Tailwind CSS

### Backend
- Node.js
- Express.js
- MongoDB
- JWT Authentication

### AI Integration
- Google Gemini API (for book outline & chapter generation)

### DevOps
- Docker
- GitHub Actions (CI/CD)
- AWS EC2 (deployment)
- Nginx (frontend hosting)

## 🧠 Features

- 🔐 User authentication (JWT)
- 📖 AI-generated book outlines
- ✍️ AI chapter generation
- 🧾 Book editing system
